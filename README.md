# HanPi 百宝箱

AI搭建 · 纹理生成、预览、效果小工具（GitHub Pages 静态站点）。

**入口首页 → [https://ashenoneart.github.io/hanpi-toolbox/](https://ashenoneart.github.io/hanpi-toolbox/)**

## 工具

- [Flowmap 生成](https://ashenoneart.github.io/hanpi-toolbox/flowmap/) — **单图生成 Flowmap** 并预览扭曲；内置示例图 `flowmap/examples/sample-source.png`；附加双图导入预览

## 本地预览

任意静态服务器即可，例如：

```bash
npx serve .
```

## 说明

贴图仅在浏览器本地处理，不会上传到服务器。
